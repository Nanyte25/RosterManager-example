# Installation


### Install meteor


```
curl https://install.meteor.com/ | sh

```



### git clone the repo

```
git clone https://github.com/Nanyte25/RosterManager.git

```


### install packages

```

meteor npm install --save bcrypt

meteor --allow-incompatible-update

meteor npm install --save babel-runtime simpl-schema


```
